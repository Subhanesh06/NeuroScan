Workflow

Built for the full issue lifecycle

01

Citizen submits evidence

Users sign in with a verified account, capture an image, and drop a precise pin on the map.

02

AI classifies the issue

The uploaded image is processed through the detection pipeline to estimate issue type and confidence.

03

Authorities take action

Admins review the report, update the status, and coordinate field work using the attached evidence.
